# QA Automation Project - Cypress

Projeto de automação de testes utilizando Cypress.

## Objetivo

Demonstrar habilidades em automação de testes end-to-end para aplicações web.

## Ferramentas utilizadas

- Cypress
- JavaScript
- Node.js
- Git
- GitHub

## Cenários de teste

- Login válido
- Login inválido
- Adicionar produto ao carrinho
- Checkout de compra

## Aplicação testada

https://www.saucedemo.com

## Como executar o projeto

1 Instalar dependências "npm install"
2 Executar testes "npx cypress open" ou "npx cypress run"
